# Tents and Trees - Version imprimable (en LaTeX)

## Compilation

Les fichiers `classe-tex3R.cls`, `style-tex3R.sty` et `flashcards-tex3R.sty` sont des fichiers de configuration personnels. Leurs versions récentes peuvent être trouvées dans ce dépôt : [https://github.com/Tex3rivieres/TeX3R-ClasseStyle/releases](https://github.com/Tex3rivieres/TeX3R-ClasseStyle/releases)

Le fichier `TnT.lua` contient le code nécesaire pour générer algorithmiquement les grilles de Tent & Trees.

Le fichier .tex doit être compilé en lualatex. Les dépendances (fichiers `.cls `et `.sty` personnels, images `.png`, script `.lua`) doivent être à la racine du fichier.
